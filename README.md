# Customer Churn Analysis (Excel)

## Project Overview

This project analyzes customer churn data to identify patterns that contribute to customer attrition. The goal of the analysis is to understand which customer segments are most likely to churn and to provide insights that can help businesses improve customer retention.
The analysis was performed using Microsoft Excel through data cleaning, pivot table analysis, and dashboard visualization.


## Tools Used
* Microsoft Excel


## Project Objectives
* Analyze customer churn behavior
* Identify high-risk customer segments
* Calculate key churn metrics
* Create an interactive dashboard to visualize churn patterns


## Data Analysis Process

### Data Preparation

The dataset was cleaned and prepared in Excel by:
* Checking for missing values
* Ensuring correct data types
* Organizing the dataset for analysis

### Exploratory Analysis
Pivot tables were used to analyze churn patterns across different variables such as:
* Customer tenure
* Payment behavior
* Support calls
* Risk category

### Risk Segmentation
Customers were segmented into 'Low, Medium, and High risk categories' based on behavioral patterns related to churn.

### Dashboard Development
An Excel dashboard was created to display key metrics and insights using charts, KPIs, and filters.



## Key Insights
* The overall churn rate in the dataset is approximately 47%.
* High-risk customers represent a large share of total churn.
* Customer behavior indicators such as payment delays and support interactions show strong relationships with churn likelihood.
